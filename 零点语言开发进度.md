## 安装与使用

### 代码生成器使用
```bash
# 输出指定值
python -m zero_lang.generator output <value>

# 计算a+b并输出
python -m zero_lang.generator add <a> <b>

# 计算a-b并输出
python -m zero_lang.generator sub <a> <b>

# 计算a*b并输出
python -m zero_lang.generator mul <a> <b>

# 计算a/b并输出
python -m zero_lang.generator div <a> <b>

# 输入并输出
python -m zero_lang.generator input-output
```

### 编译器使用（即将推出）
```bash
# 编译为Windows x86_64可执行文件
zero-compiler --platform windows-x86_64 <input_file> -o <output.exe>

# 编译为Linux x86_64可执行文件
zero-compiler --platform linux-x86_64 <input_file> -o <output>

# 编译为ARM64可执行文件
zero-compiler --platform arm64 <input_file> -o <output>

# 编译为单片机固件
zero-compiler --platform cortex-m <input_file> -o <output.bin>
```
